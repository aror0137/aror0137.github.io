HTML Tag Glossary

<nav>
A tag used to define a set of navigation links.
<img>
Used to embed an image in a web page.
<ul>
Represents an unordered list of items, typically rendered as a bulleted list.
<dl>
The description list tag, used to enclose terms and their descriptions.
<dt>
The description term tag, used to specify the name or term in a list.
